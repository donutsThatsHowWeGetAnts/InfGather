# InfGather

To install:

  * Download repository
  * cd InfGather/bash
  * ./infGather -p


### Dependencies

  * nmap
  * enum4linux
  * dirb
  * nikto
  * zentiy || xmessage
  
  Open to suggestions, let me know if there is anything else that should be added/modified!
